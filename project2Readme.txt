Exploratory Data Analysis (EDA)

Overview
This project focuses on performing Exploratory Data Analysis (EDA) on an ecommerce dataset using Python, Pandas, and Matplotlib. The objective was to identify trends, correlations, revenue patterns, and outliers through statistical analysis and visualization techniques.

Tools & Technologies
- Python
- Pandas
- Matplotlib

Tasks Performed
- Loaded and explored the cleaned dataset
- Generated statistical summaries using describe()
- Analyzed product-wise sales performance
- Evaluated payment method usage
- Analyzed monthly revenue trends
- Performed correlation analysis
- Detected outliers using the Interquartile Range (IQR) method
- Created visualizations for revenue analysis

Key Findings
- Printer had the highest number of orders.
- Chair generated the highest overall revenue.
- Online payment was the most frequently used payment method.
- Month 6 recorded the highest sales revenue.
- Strong positive correlation was observed between UnitPrice and TotalPrice.
- Several high-value outlier transactions were identified using the IQR method.

Visualization
A horizontal bar chart was created to visualize revenue generated by each product category.
